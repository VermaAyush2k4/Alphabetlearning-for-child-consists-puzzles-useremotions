Process
  • Divide alphabets into set 5. For every iteration (try) the child would engage with all 5 activities for respective letter.
  • Once the child completes a single iteration, hand it a physical (paper) test or tracer to remember all the letters and it's associated terms.
  • Note down the observation. Repeat above steps for next set of 5 letters and so on for all 26.
  • Now compare observations after each set and mark down final observation 1 and result.
  • Rerun entire process for again 26 alphabets and mark down final observation 2. Compare with preview final observation and reach the conclusion.
